# Support

## Community Support

- **Documentation:** See [README.md](./README.md) for project overview, [ARCHITECTURE.md](./ARCHITECTURE.md) for technical details, and [DEVELOPER_GUIDE.md](./DEVELOPER_GUIDE.md) for development setup.
- **FAQ:** Visit the [FAQ page](https://dev-mon.netlify.app/faq) for answers to common questions.

## Reporting Issues

- **Bug reports:** Open a [GitHub issue](https://github.com/ShravanDeb/DevMon/issues) with a clear description and reproduction steps.
- **Feature requests:** Open a [GitHub issue](https://github.com/ShravanDeb/DevMon/issues) with the `[Feature]` prefix.
- **Security vulnerabilities:** Do not open a public issue. See [SECURITY.md](./SECURITY.md) for responsible disclosure.
